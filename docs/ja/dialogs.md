# ダイアログ一覧

## 設定ダイアログ

- エディタフォントサイズ、各種色指定などを行う。
- Other > Settings... F8 で表示される

![画像](image_20260303_074224.png)

- エディタ部分、プレビュー部分、全体的な部分に分かれる。
- エディタ部分：
  - フォントサイズ：
Markdown文書のフォントサイズを指定します。
  - タイトル/見出しテキスト色：
Markdownの # タイトル/見出し表示色を指定します。
  - ボールドテキスト色
Markdownの **ボールド** テキスト表示色を指定します。
  - イタリックテキスト色
Markdownの *イタリック* テキスト表示色を指定します。
  - 打ち消し線テキスト色
Markdownの ~~打ち消し線~~ テキスト表示色を指定します。
  - 検索マッチ強調背景色
Markdownの検索にマッチしたテキスト表示色を指定します。
- プレビュー部分：
  - CSVヘッダ背景色
  - CSVブロックゼブラカラー（偶数行）
  - CSVブロックゼブラカラー（奇数行）
  - 引用ブロック背景色
  - コードブロック背景色
  - 罫線ブロック背景色
- 全体部分
  - アクティブカーソル色
  - 非アクティブカーソル色
