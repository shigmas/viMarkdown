## Key Features
- **Real-time Editor-Preview Synchronization**
  The cursor position and edits are synchronized seamlessly in real time. This allows for intuitive editing while fully preserving the structural integrity of your Markdown documents.
- **Direct Editing in the Preview Pane**
  Allows inserting and deleting text directly within the preview, not just the editor. You can complete your writing in a true WYSIWYG (What You See Is What You Get) manner.
- **Integrated Vi Keybindings (Vi Commands)**
  Full support for standard vi commands, enabling rapid text editing and cursor navigation without ever having to move your hands from the home row.
- **Powerful Grep (Global Search) Functionality**
  Instantly extract target text across multiple files or entire directories, strongly supporting efficient document search and bulk editing.
- **Custom Blocks & SVG Rendering Support**
  In addition to text-based "Keisen" (ruled line) and "CSV" blocks, it features built-in support for rendering SVG, allowing you to represent diagrams directly via code.
- **Native Implementation with Qt6/C++**
  Unlike Electron-based text editors, this app is built natively using C++ and Qt6, delivering lightweight performance and outstanding responsiveness.
- **Smart Keisen (Ruled Line) Table Protection**
  Features a unique algorithm that automatically aligns text-based table borders while you type, preventing layouts from collapsing.
