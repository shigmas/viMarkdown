# vi の名を冠するマークダウンエディタを
# 忘年会のノリで開発開始してしまった話

## アウトライン

- 以前からなんとなくマークダウンエディタを開発してみたかった
- 実は20年以上前に、行頭記号記法方式アプリ（vine）を開発していた
  - ■ で始まる行は大見出し、● で始まる行は中見出し、○ で始まる行は小見出し みたいな仕様
  - 行頭記号記法で記述したソーステキストを流し込むと綺麗にレイアウトしてくれる
  - メジャーにはならなかった（英語環境では使用するのが困難？）
  - 完成度を上げれなかった？
  - マークダウン書式って行頭記号記法に一部似てる
  - マークダウン書式がある程度（特にエンジニア間では）メジャーになりちょっとくやしい
- 2025/12/11 のテニスサークル忘年会の席上で某元助教授（原子核工学）Ｋ氏から唐突に何かコラボしようと誘われた
  - 忘年会の前にはテニスサークルの試合があったのだが、Ｋ氏は前日に原稿を徹夜で書いていたとのことだった（そのせいで電車で寝込んでしまい忘年会に遅刻した）
  - 具体的に何のコラボするかは話が進まず
  - 何か研究者が使用するアプリを開発するのでＫ氏が監修者になると提案した気がする
- 次の日に、コラボとして何をするかいろいろ考え、とりあえずマークダウンエディタを作ってみるかという気になった。12/13 から開発開始
- アプリ名を何にするか色々迷った末、viMarkdown というアプリ名にした
- その結果 vi コマンドを実装せざるとえないことになってしまった
- この原稿執筆時（2026/01下旬）で多くの機能を実装・動作確認し、2月にはアルファ版をリリース予定
- Ｋ氏にアプリを見せて、感想を聞きたいのだが、その後Ｋ氏とは会っていない

- viMarkdown はソースを https://github.com/vivisuke/viMarkdown で公開している
- Windows 版バイナリもリリースしてる（個人利用は無料）ので、興味ある方はDLして試してみてね
